# Markdown Features Showcase

Markdown is a lightweight markup language for creating formatted text using a plain-text editor. Here's a showcase of various features you can use in Markdown.

---

## 1. Headers

# H1 Header
## H2 Header
### H3 Header
#### H4 Header
##### H5 Header
###### H6 Header

---

## 2. Emphasis

- *Italic* text with asterisks
- _Italic_ text with underscores
- **Bold** text with double asterisks
- __Bold__ text with double underscores
- ~~Strikethrough~~ text with tildes

---

## 3. Lists

### Unordered List:
- Item 1
- Item 2
  - Nested Item 1
  - Nested Item 2
    - Nested again

### Ordered List:
1. First item
2. Second item
   1. Sub-item 1
   2. Sub-item 2
